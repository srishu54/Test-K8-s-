# Test-K8-s-
Test-K8-s-
This repository contains a sample Docker project demonstrating:

Building a RHEL 9-based Docker image running Apache HTTPD.
Hosting a sample web page (index.html) inside the container.
Integrating the Docker build with Jenkins.
Deploying the Docker image as a pod on a Kubernetes cluster.
This project serves as a DevOps/Kubernetes test project for CI/CD workflows.
